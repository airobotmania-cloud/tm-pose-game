# 게임 규칙 (GAME_RULE.md)

## 🎮 게임 제목
**Fruit Catcher (과일 받기 게임)**

---

## 📝 게임 설명
하늘에서 떨어지는 과일을 바구니로 받아내는 게임입니다.
내 몸을 움직여 바구니를 왼쪽, 가운데, 오른쪽으로 이동시키세요!
폭탄을 받으면 게임이 종료되니 조심하세요.

---

## 🕹️ 게임 규칙

### 기본 구조
* 3개의 구역(LEFT, CENTER, RIGHT)으로 나뉩니다.
* 플레이어의 포즈에 따라 바구니가 해당 구역으로 이동합니다.

### 조작 방법 (Teachable Machine 모델 클래스)
| 포즈 이름 (Label) | 설명 | 바구니 위치 |
|:---:|:---:|:---:|
| **LEFT** | 왼쪽으로 기울이거나 손을 왼쪽으로 뻗음 | 왼쪽 구역 (Zone 1) |
| **CENTER** | 정면을 보고 차렷 자세 | 가운데 구역 (Zone 2) |
| **RIGHT** | 오른쪽으로 기울이거나 손을 오른쪽으로 뻗음 | 오른쪽 구역 (Zone 3) |

## ⚙️ 상세 설정
1. **제한 시간**: 60초
2. **속도 증가**: 시간이 지날수록 과일이 떨어지는 속도가 빨라집니다.
3. **게임 오버 조건**:
   - 폭탄을 받았을 때
   - 제한 시간이 모두 지났을 때
